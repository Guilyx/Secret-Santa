# Secret Santa

For better use, write a shell script as followed :

```
python3 main.py << EOF
4 # number of participants
John # names of the participants (more or less than 4 inputs not supported yet)
Doe
Foo
Bar
email1@example.com
email2@example.com
email3@example.com
email4@example.com
sender-email@example.com
sender-password
EOF
```
